# Test Bundle

This is a test bundle used by the Capacium Test Lab.
It validates bundle install, verification, and lifecycle across all supported agent frameworks.

## Contents

This bundle contains two sub-capabilities:

- **sub-skill**: A test skill for bundle testing
- **sub-tool**: A test tool for bundle testing

## Usage

When installed, this bundle provides no functional capabilities.
It exists solely for automated testing of the Capacium bundle system.
